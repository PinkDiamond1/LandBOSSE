# LandBOSSE

This is a design-based land-based wind power plant balance of system cost model.

Author: [A. Eberle](annika.eberle@nrel.gov)

## Prerequisites

Python, NumPy, SciPy

## Installation

Install LandBOSSE with the following command.

    $ python setup.py install
